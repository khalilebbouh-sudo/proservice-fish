# Full Stack Web Application

Application web complète avec dashboard admin et interface utilisateur.

## Features
- Authentification utilisateur
- Gestion des données (CRUD)
- Dashboard administrateur
- API REST

## Technologies
- Frontend : Angular / Next.js
- Backend : Node.js / Spring Boot
- Base de données : MongoDB / MySQL

## Objectif
Projet réalisé pour améliorer mes compétences en développement full stack et en conception d’applications web modernes.
